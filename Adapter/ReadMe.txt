Паттерн Адаптер (Adapter) 
предназначен для преобразования интерфейса одного класса в интерфейс другого. 
Благодаря реализации данного паттерна мы можем использовать вместе классы с несовместимыми интерфейсами.

Когда надо использовать Адаптер?

	 - Когда необходимо использовать имеющийся класс, но его интерфейс не соответствует потребностям
	 - Когда надо использовать уже существующий класс совместно с другими классами, 
	 интерфейсы которых не совместимы

Задача на самостоятельную работу
Создать адаптер квадрата к кругу. Адаптер должен возвращать квадрат что описывает заданный круг.
